Tecnologia Spring MVC


Este é um pequeno projeto que demonstra o uso de uma aplicação desenvolvida em Spring MVC

Foram utilizadas as seguintes ferramentas:

- Thymeleaf
- Spring Boot
- Spring Data
- Spring Security
- Maven
- PostgreSQL
- Heroku(Deploy)
- Spring Tool Suite(IDE)
